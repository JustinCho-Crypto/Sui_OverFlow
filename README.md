# Charui - SUI 기반 기부 플랫폼

퉁퉁퉁퉁퉁퉁퉁퉁퉁퉁 사호르
트랄라렐롤 트랄랄라
SUI 블록체인을 활용한 기부 플랫폼입니다.

## 주요 기능

- SUI 코인을 통한 기부
- 다양한 카테고리별 기부
- 기부 내역 추적
- 지갑 연결 및 트랜잭션 처리

## 기술 스택

- Frontend: Next.js, TypeScript
- Blockchain: SUI
- Wallet: @mysten/dapp-kit

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
